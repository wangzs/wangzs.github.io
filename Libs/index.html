1、Android开发库

1.1 设计模式相关

  • mosby: A Model-View-Presenter library for modern Android apps

1.2 网络相关

  • retrofit: Square公司出品的HTTP client库，非常好用的http库(构建与okhttp之上)
  • Android-Download-Manager-Pro: 下载管理的库

1.3 数据库相关

  • greenDAO: Light & fast ORM solution for Android that maps objects to SQLite databases
  • hawk: 方便的数据本地存储和读取（可以设置存储为sp或sqlite）
  • realm-java: Realm is a mobile database: a replacement for SQLite & ORMs.

1.4 UI相关

  • Fancybuttons: 不用配置selector也能做出好看按钮的库
  • RippleEffect: 为低版本提供MD的波纹点击效果
  • slidingtutorial-android: 快速的制作应用首次进入的向导库
  • uCrop: 好用的裁剪图像的库
  • StaggeredGridView: 类似Pinterest的瀑布流显示效果的gridview
  • AndroidStaggeredGrid: 类似Pinterest的瀑布流显示效果的gridview(关注度更高)
  • material-dialogs: A beautiful, easy-to-use, and customizable dialogs API
  • cardslib: Provides an easy way to display a UI Card using the Official Google CardView in your Android app
  • MaterialDrawer: 符合MD的侧边栏，比官方的drawer用起来方便、集成快
  • app-theme-engine: 主题切换库
  • ViewPagerIndicator: 有指示的ViewPager库（可方便自定义指示标志）
  • SwipeMenuListView: 还算好用的自定义侧滑item操作的库
  • dialogplus: 弹出对话框辅助库（提供底、顶、中间三种位置弹出；提供Listview、Gridview或自定义的显示样式）
  • RoundedImageView: A fast ImageView that supports rounded corners, ovals, and circles.
  • CircleImageView: 产生类似圆形轮廓头像的ImageView
  • ExplosionField: 花哨的爆炸消失的特效库
  • fresco: Facebook出品的强大的图片显示库（Fresco takes care of image loading and display）
  • picasso: Square公司出品的图片加载和显示的库（本地、网络、资源id都可以用来显示）
  • Context-Menu.Android: 好看的点击后弹出菜单的库（Yalantis公司出品了很多好看动画效果UI控件）
  • PhotoView: Implementation of ImageView for Android that supports zooming, by various touch gestures.
  • Android-PullToRefresh: 支持下拉上拉操作（star最多）

1.5 工具库

  • leakcanary: 开始过程中检查程序的内存泄漏的神器
  • logger: 很好用的logcat打印库(可以查看到log的所在线程、触发代码位置等)
  • SuperToasts: 可以理解为超级Toast，各种弹都没问题
  • guava: Google的java核心库（集合、并发库、缓存、注解、string处理、I/O等）
  • RxJava: RxJava有学习成本，掌握后会极大解放生产力（回调神马不再各种嵌套、更新UI等各种好处）
  • EventBus: 简化Activities, Fragments, Threads, Services等之间的数据传递

1.6 注入相关

  • Dagger 2: A fast dependency injector for Android and Java.（google在square的dagger基础上的修改版本）
  • butterknife: Android下简化代码，解放生产力的注入库

1.7 AS插件

  • android-selector-chapek: 将一定命名规则的图片自动生成selectors的AS插件
  • ADB Idea: ADB辅助工具插件
  • android-parcelable-intellij-plugin: AS的parcelable模板代码生成插件
